# Mission-to-Mars

## Purpose of the project

To build a flask app to scrape the latest news about mars mission and get the images related to mars from  the website,

## Resources:

- Flask
- Python 3.7.6, Flask 1.1.2, JupyterLab 2.26, VS Code 1.51.1
- MongoDB 4.4.2
- HTML5, Bootstrap 3
- Beautiful Soup
- Splinter
